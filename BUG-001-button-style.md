# BUG-001: Кнопка на странице "Авторы" не соответствует дизайну

## Описание
Кнопка на странице "Авторы" (для роли библиотекарь) визуально отличается от остальных кнопок приложения и не соответствует единому стилю интерфейса. Связанный тест-кейс
TC-008 — Проверка соответствия кнопок дизайну

## Окружение
- ОС: Windows 11
- Тип приложения: WPF
- Роль: Библиотекарь

## Предусловие
Пользователь авторизован под ролью "Библиотекарь"

## Шаги воспроизведения
1. Перейти в раздел "Авторы" 
2. Найти кнопки управления (Добавить / Редактировать / Удалить)
3. Сравнить внешний вид кнопок с другими страницами приложения

## Ожидаемый результат
Кнопка соответствует дизайну приложения:
- одинаковый цвет #FF2C3E50 фона кнопки и надписей #FFFFFFFF
- единый шрифт Franklin Gothic Book
- одинаковый размер 16px

## Фактический результат
Кнопка "Добавить" отличается от остальных элементов интерфейса:
- отличается цвет / размер / стиль
